# 神经网络（Neural Network）

> 把简单的计算单元一层层叠起来，组成能学习复杂规律的函数

## TL;DR

**神经网络**是深度学习的基础架构：

- **神经元**: 加权求和 + 激活函数（最小计算单元）
- **激活函数**: 让直线变弯，使网络能表达复杂关系（最常用 ReLU）
- **层**: 多个神经元并排组成一层，多层叠起来组成网络
- **深度学习**: 层多的神经网络，逐层从简单到复杂提取特征
- **变体**: CNN（擅长图像）、RNN（擅长序列）、Transformer（当前主流）

---

## 1. 神经元 = 加权求和 + 激活函数

### 加权求和

不同输入的重要程度不一样，乘以不同的权重再加起来：

```text
选房子考虑三个因素：

  面积 × 3（很重要）+ 卧室数 × 1（一般）+ 距离 × 0.5（不太重要）
  = 120×3 + 3×1 + 2×0.5
  = 364

"加权" = 有的乘大数有的乘小数
"求和" = 加起来
```

模型"学习"学的就是这些权重应该是多少。

### 激活函数

加权求和的结果再过一道"激活函数"，让直线变弯。没有激活函数，不管叠多少层都是直线：

```text
无激活函数：
  第1层: y = 2x + 1
  第2层: z = 3y + 4 = 3(2x+1) + 4 = 6x + 7
  → 两层 = 一层，永远是直线，叠再多也没用

有激活函数：
  第1层: y = ReLU(2x + 1)
  第2层: z = ReLU(3y + 4)
  → 能表达弯曲、折线、复杂的形状了
```

---

## 2. ReLU（最常用的激活函数）

读作 **"瑞露"**（/ˈrelʊ/），全称 Rectified Linear Unit（修正线性单元）。

规则极其简单：**负数变 0，正数不变。**

```text
ReLU(-3) = 0
ReLU(0)  = 0
ReLU(5)  = 5
```

### ReLU 怎么"变弯"？

一个 ReLU 把直线砍一刀变成折线，多个 ReLU 叠加就能逼近任意曲线：

```text
神经元A：在 x=2 处砍一刀 → 一个折点
神经元B：在 x=5 处砍一刀 → 又一个折点

A + B 叠加 → 两个折点，斜率在 x=2 和 x=5 处变化

3 个神经元  → 粗糙的折线
100 个神经元 → 密到看起来像光滑曲线
```

**每多一个神经元就多砍一刀，砍得越多越能逼近任意形状。**

## 3. 层（Layer）

### 网络结构

```text
输入层        隐藏层         隐藏层         输出层
(特征)       (自动提取)     (进一步提取)     (结果)

 x₁ ──→  ●  ──→  ●  ──→  ●
              ╲╱          ╲╱
 x₂ ──→  ●  ──→  ●  ──→  ● ──→ 输出
              ╱╲          ╱╲
 x₃ ──→  ●  ──→  ●  ──→  ●

每个 ● = 一个神经元
每条线 = 一个权重参数
```

### 逐层提取特征

每一层把上一层的简单发现组合成更复杂的发现：

```text
图像识别中：
  第1层：边缘、线条
  第2层：纹理、角、简单形状
  第3层：眼睛、耳朵、鼻子
  第4层：整张脸
```

---

## 4. 深度学习 = 层多的神经网络

```text
浅层神经网络: 输入 → 1~2层 → 输出    ← 传统神经网络
深度神经网络: 输入 → 几十上百层 → 输出 ← 深度学习
```

层越多，能学到的模式越复杂，但也需要更多数据和算力。

---

## 5. 神经网络的变体

| 变体 | 连接方式 | 擅长 |
| --- | --- | --- |
| **全连接网络** | 每个神经元连接上一层所有神经元 | 表格数据 |
| **CNN** | 用固定大小的卷积核滑窗扫描 | 图像 |
| **RNN** | 带记忆，逐步处理序列 | 语音、文本 |
| **Transformer** | 用 Attention 全局连接 | 当前主流（GPT、BERT） |

它们都是神经网络，只是连接方式不同。详见 [[序列模型]]。

### 和之前知识的对应

| 之前学的概念 | 其实就是 |
| --- | --- |
| `y = w×x + b`（线性回归） | 一个没有激活函数的单个神经元 |
| 参数 w | 神经元之间连线上的权重 |
| "学习" | 调整所有权重，让损失函数最小 |
