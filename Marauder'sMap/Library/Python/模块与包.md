# 模块与包

## TL;DR

**Python 用模块（.py 文件）和包（含 `__init__.py` 的目录）来组织代码。** 核心要点：

- 一个 `.py` 文件就是一个模块，模块通过 `import` 导入
- 包是含有 `__init__.py` 的目录，用来避免模块名冲突
- `__init__.py` 可以为空，也可以写初始化代码、控制导出
- 模块只在**第一次 import 时执行**，之后从 `sys.modules` 缓存读取

---

## 1. 模块（Module）

一个 `.py` 文件就是一个模块：

```text
abc.py  → 模块名 abc
xyz.py  → 模块名 xyz
```

导入和使用：

```python
import sys
sys.argv          # 访问模块内的变量

from collections.abc import Iterable   # 导入模块中的某个名字
```

---

## 2. 包（Package）

为了避免模块名冲突，用目录组织模块，目录下必须有 `__init__.py`：

```text
mycompany/
├── __init__.py    ← 必须有，否则只是普通目录
├── abc.py         ← 模块名：mycompany.abc
└── xyz.py         ← 模块名：mycompany.xyz
```

可以有多级目录：

```text
mycompany/
├── __init__.py
├── abc.py
├── utils.py       ← mycompany.utils
└── web/
    ├── __init__.py
    ├── utils.py   ← mycompany.web.utils
    └── www.py     ← mycompany.web.www
```

---

## 3. `__init__.py` 能写什么

`__init__.py` 本身就是一个模块，它的模块名就是**包名**（如 `mycompany`）。可以为空，也可以写代码：

| 用途 | 示例 |
|------|------|
| 控制 `*` 导出 | `__all__ = ['abc', 'xyz']` |
| 包级别初始化 | `print('初始化')` |
| 便捷导入 | `from .abc import some_func` |
| 包级别变量 | `__version__ = '1.0.0'` |

最常见的用法是**提升子模块接口到包级别**：

```python
# mycompany/__init__.py
from .abc import some_function

# 外部可以直接：
from mycompany import some_function
# 而不需要：
from mycompany.abc import some_function
```

`flask`、`requests` 等知名库都是这么做的。

---

## 4. import 只执行一次

模块代码只在**第一次 import 时执行**，后续 import 从缓存读取：

```python
# mycompany/__init__.py
print('初始化!')

# main.py
import mycompany   # 打印 "初始化!"
import mycompany   # 什么都不打印
import mycompany   # 什么都不打印
```

已导入的模块缓存在 `sys.modules` 中：

```python
import sys
import mycompany
'mycompany' in sys.modules  # True
```

需要强制重新加载（调试时）：

```python
import importlib
importlib.reload(mycompany)  # 再次执行模块代码
```

---

## 5. 作用域

Python 用 `_` 前缀约定访问权限：

| 命名 | 含义 | 示例 |
|------|------|------|
| `abc` / `x123` | 公开（public） | 正常使用 |
| `__xxx__` | 特殊变量 | `__name__`、`__author__` |
| `_xxx` / `__xxx` | 私有（private） | 不应该被外部引用 |

注意：Python 的 private 只是约定，**不是强制**。技术上仍然可以访问，但不应该这么做。

---

## 6. `if __name__ == '__main__'`

```python
if __name__ == '__main__':
    test()
```

- 命令行直接运行 `python hello.py` 时，`__name__` 是 `'__main__'`，执行 `test()`
- 被其他模块 `import hello` 时，`__name__` 是 `'hello'`，不执行

这是让模块既能被导入使用，又能独立运行测试的标准写法。
