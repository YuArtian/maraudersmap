# 函数进阶

## TL;DR

**Python 函数是一等对象，可以赋值、传参、返回，还有 lambda 匿名函数和装饰器等高级用法。** 核心要点：

- `lambda` 是一行匿名函数，等价于 JS 的箭头函数
- Python 函数有两种类型：`def` 定义的 `FunctionType` 和 C 实现的 `BuiltinFunctionType`
- Python 没有 `const`，常量靠全大写命名约定
- `callable()` 判断对象能不能调用，比检查类型更实用

---

## 1. lambda 匿名函数

`lambda` 就是一行写完的匿名函数，自动返回表达式结果：

```python
# 这两个完全等价
def add(x, y):
    return x + y

add = lambda x, y: x + y
```

对比 JS 箭头函数：

| Python | JavaScript |
|--------|-----------|
| `lambda x: x * 2` | `x => x * 2` |
| `lambda x, y: x + y` | `(x, y) => x + y` |
| `lambda: 42` | `() => 42` |

最常见的用途是传给高阶函数：

```python
sorted(['bob', 'Alice'], key=lambda s: s.lower())

list(map(lambda x: x * x, [1, 2, 3]))       # [1, 4, 9]

list(filter(lambda x: x > 0, [-1, 2, -3]))  # [2]
```

限制：lambda 只能写**一个表达式**，不能写多行语句。

## 2. 函数的两种类型

Python 中 `def` 定义的函数和内置函数类型不同：

```python
import types

def my_func():
    pass

type(my_func)  # <class 'function'>
type(abs)      # <class 'builtin_function_or_method'>
```

| 类型 | 来源 | 示例 |
|------|------|------|
| `types.FunctionType` | `def` / `lambda` 定义 | `my_func`, `lambda x: x` |
| `types.BuiltinFunctionType` | C 语言实现的内置函数 | `abs`, `len`, `print` |

```python
type(my_func) == types.FunctionType         # True
type(abs) == types.FunctionType             # False
type(abs) == types.BuiltinFunctionType      # True
```

注意：`types.LambdaType is types.FunctionType` 为 `True`，lambda 和 def 创建的是同一种类型。

### 用 callable() 更实用

如果只想判断"能不能调用"，不用区分类型：

```python
callable(abs)      # True
callable(my_func)  # True
callable(123)      # False
```

## 3. 装饰器中的 `*args, **kw`

装饰器用 `*args, **kw` 让 wrapper 能接收并透传任意参数：

```python
def log(func):
    def wrapper(*args, **kw):    # 接收任意参数
        print('call %s():' % func.__name__)
        return func(*args, **kw) # 原封不动传给原函数
    return wrapper
```

```text
调用端:  wrapper(1, 2, name='Bob')
               ↓ 打包
wrapper 内部:  args=(1, 2)  kw={'name': 'Bob'}
               ↓ 解包
转发:    func(1, 2, name='Bob')
```

这是固定套路：写装饰器/包装函数时，`*args, **kw` 接收，再用 `*args, **kw` 转发。

## 4. 常量约定

Python **没有 `const`**，靠全大写命名表示常量：

```python
PI = 3.14159
MAX_RETRY = 3
DATABASE_URL = 'localhost:5432'
```

但不会阻止修改：

```python
PI = 100  # 没有报错，Python 不管
```

| | Python | JavaScript |
|---|---|---|
| 常量声明 | 无，靠命名约定 | `const x = 1` |
| 能阻止修改吗 | 不能 | 能 |
| 习惯写法 | `MAX_SIZE = 100` | `const MAX_SIZE = 100` |

Python 的设计哲学："We're all consenting adults" -- 靠自觉而不是语言强制。
