# 字符串格式化

## TL;DR

**Python 有三种字符串格式化方式**，从老到新：`%` 格式化、`.format()`、f-string。推荐用 f-string（Python 3.6+），最简洁直观。三种方式语法不同但效果一样，注意 `%` 写法没有冒号，`.format()` 和 f-string 用冒号分隔格式说明。

---

## 1. 三种写法对比

同样的效果，三种写法：

```python
name = '小明'
score = 17.125

# % 格式化（最老）
'Hello, %s, 成绩提升了 %.1f%%' % (name, score)

# .format()
'Hello, {}, 成绩提升了 {:.1f}%'.format(name, score)

# f-string（推荐）
f'Hello, {name}, 成绩提升了 {score:.1f}%'

# 结果都是：'Hello, 小明, 成绩提升了 17.1%'
```

---

## 2. % 格式化

语法：`%` + 格式说明符

| 说明符 | 含义 | 例子 | 结果 |
|--------|------|------|------|
| `%s` | 字符串（万能，任何类型都能转） | `'%s' % 42` | `'42'` |
| `%d` | 整数 | `'%d' % 42` | `'42'` |
| `%f` | 浮点数 | `'%f' % 3.14` | `'3.140000'` |
| `%.1f` | 保留 1 位小数 | `'%.1f' % 17.125` | `'17.1'` |
| `%%` | 输出字面 `%` 号 | `'100%%'` | `'100%'` |

`%s` 万能——Python 的一切对象都能转成字符串（都有 `__str__` 方法），所以 `%s` 对任何东西都能用。

---

## 3. .format()

语法：`{位置:格式说明}` ，冒号左边是"取第几个参数"，右边是"怎么显示"：

```python
'{1:.1f}'.format('小明', 17.125)
  ↑ ↑↑
  │ │└── 保留 1 位小数的浮点数
  │ └─── 冒号，分隔位置和格式
  └───── 取第 1 个参数（17.125）
```

位置可以省略，按顺序自动对应：

```python
'Hello, {}, 成绩提升了 {:.1f}%'.format('小明', 17.125)
```

---

## 4. f-string（推荐）

Python 3.6+ 支持，字符串前加 `f`，直接在 `{}` 里写变量名：

```python
name = '小明'
score = 17.125
f'Hello, {name}, 成绩提升了 {score:.1f}%'
```

最直观，不用数位置，变量名就在字符串里。

---

## 5. 格式说明符对比

两套写法的格式说明符不同，注意 `%` 写法**没有冒号**：

| 效果 | `%` 写法 | `.format()` / f-string |
|------|----------|----------------------|
| 字符串 | `%s` | `{}` |
| 整数 | `%d` | `{:d}` |
| 浮点数 | `%f` | `{:f}` |
| 保留 1 位小数 | `%.1f` | `{:.1f}` |
| 保留 2 位小数 | `%.2f` | `{:.2f}` |

`%` 写法用 `%` 作为前缀直接跟说明符；`.format()` 用 `{}` 包裹，冒号后跟说明符。
