# 函数参数

## TL;DR

**Python 函数支持五种参数形态，从简单到复杂可以灵活组合。** 核心要点：

- `*args` 接收可变位置参数（tuple），`**kw` 接收关键字参数（dict）
- 默认参数必须用不可变对象，否则会产生隐蔽 bug
- 命名关键字参数用 `*` 分隔符限制调用者可传入的参数名
- 五种参数可以组合使用，但顺序有严格规定

---

## 1. 五种参数类型一览

| 类型 | 语法 | 接收的数据 | 示例定义 |
|------|------|-----------|---------|
| 位置参数 | `def f(x, y)` | 按位置传入 | `f(1, 2)` |
| 默认参数 | `def f(x, y=10)` | 有默认值，可省略 | `f(1)` 或 `f(1, 20)` |
| 可变参数 | `def f(*args)` | 一个 tuple | `f(1, 2, 3)` |
| 关键字参数 | `def f(**kw)` | 一个 dict | `f(a=1, b=2)` |
| 命名关键字参数 | `def f(*, name)` | 指定名称的参数 | `f(name='Bob')` |

---

## 2. 默认参数的坑：必须用不可变对象

```python
# 错误写法 -- 默认参数用了可变对象 list
def add_end(L=[]):
    L.append('END')
    return L

add_end()  # ['END']
add_end()  # ['END', 'END']  -- 不是预期的 ['END']!
add_end()  # ['END', 'END', 'END']  -- 越来越长...
```

原因：默认参数 `L` 指向的 list 对象在函数定义时就创建了，每次调用都在同一个对象上操作。

```python
# 正确写法 -- 用 None 代替
def add_end(L=None):
    if L is None:
        L = []
    L.append('END')
    return L
```

规则很简单：**默认参数只用 `None`、数字、字符串、tuple 这些不可变对象。**

---

## 3. 可变参数 `*args`

`*args` 把多余的位置参数收集成一个 **tuple**：

```python
def calc(*numbers):
    total = 0
    for n in numbers:
        total += n * n
    return total

# 直接传入
calc(1, 2, 3)        # 14

# 先组装再传入 -- 用 * 解包
nums = [1, 2, 3]
calc(*nums)           # 14
```

---

## 4. 关键字参数 `**kw`

`**kw` 把多余的关键字参数收集成一个 **dict**：

```python
def person(name, age, **kw):
    print('name:', name, 'age:', age, 'other:', kw)

# 直接传入
person('Bob', 35, city='Beijing')
# name: Bob age: 35 other: {'city': 'Beijing'}

# 先组装再传入 -- 用 ** 解包
extra = {'city': 'Beijing', 'job': 'Engineer'}
person('Bob', 35, **extra)
```

---

## 5. 命名关键字参数

如果想限制调用者只能传特定名称的关键字参数，需要用 `*` 作为分隔符：

```python
# * 后面的 city、job 就是命名关键字参数
def person(name, age, *, city, job):
    print(name, age, city, job)

person('Bob', 35, city='Beijing', job='Engineer')  # OK
person('Bob', 35, 'Beijing', 'Engineer')            # 报错! 必须用关键字传入
```

如果已经有了 `*args`，就不需要再写 `*` 分隔符：

```python
def person(name, age, *args, city, job):
    pass  # city 和 job 自动成为命名关键字参数
```

命名关键字参数也可以有默认值：

```python
def person(name, age, *, city='Beijing', job):
    pass

person('Bob', 35, job='Engineer')  # city 使用默认值
```

---

## 6. 参数组合的顺序

五种参数可以组合，但必须按以下顺序定义：

```text
位置参数 -> 默认参数 -> *args -> 命名关键字参数 -> **kw
```

```python
def f(a, b, c=0, *args, d, **kw):
    print(f'a={a}, b={b}, c={c}, args={args}, d={d}, kw={kw}')

f(1, 2, 3, 4, 5, d=99, x=100)
# a=1, b=2, c=3, args=(4, 5), d=99, kw={'x': 100}
```

---

## 7. 传参语法速查

| 场景 | 语法 | 说明 |
|------|------|------|
| 传入可变参数 | `func(1, 2, 3)` | 直接传 |
| 解包 list/tuple 传入 | `func(*[1, 2, 3])` | `*` 解包 |
| 传入关键字参数 | `func(a=1, b=2)` | 直接传 |
| 解包 dict 传入 | `func(**{'a': 1, 'b': 2})` | `**` 解包 |
