# 迭代与切片

## TL;DR

**Python 的 `for...in` 可以遍历任何可迭代对象，切片语法可以灵活截取序列片段。** 核心要点：

- `for...in` 等价于 JS 的 `for...of`，遍历的是**值**
- 切片语法 `s[start:stop:step]` 支持步长和反转
- dict 是可迭代的，有 `.keys()` / `.values()` / `.items()` 三种遍历方式
- `.values()` 等返回的是**视图对象**，不是 list，会动态反映字典变化
- `collections.abc` 提供抽象基类，用来判断对象支持哪种"协议"

---

## 1. for...in 遍历

Python 的 `for...in` 就是 JS 的 `for...of`，遍历的是**值**：

```python
# Python
for x in [1, 2, 3]:
    print(x)            # 1, 2, 3
```

```javascript
// JavaScript（等价写法）
for (const x of [1, 2, 3]) {
    console.log(x);     // 1, 2, 3
}
```

需要同时拿到索引，用 `enumerate`：

```python
for i, x in enumerate(['a', 'b', 'c']):
    print(i, x)  # 0 a, 1 b, 2 c
```

---

## 2. 切片语法

完整形式：`s[start:stop:step]`，三个参数都可以省略。

```python
s = 'ABCDEFG'

s[0:3]    # 'ABC'     -- 索引 0 到 2
s[:3]     # 'ABC'     -- 省略 start，从头开始
s[3:]     # 'DEFG'    -- 省略 stop，到末尾
s[::2]    # 'ACEG'    -- 每隔 2 个取 1 个
s[::3]    # 'ADG'     -- 每隔 3 个取 1 个
s[::-1]   # 'GFEDCBA' -- 反转
s[1:5:2]  # 'BD'      -- 从索引 1 到 5，步长 2
```

切片对 list、tuple 同样适用：

```python
L = [0, 1, 2, 3, 4, 5]
L[::2]    # [0, 2, 4]
L[::-1]   # [5, 4, 3, 2, 1, 0]
```

## 3. 遍历 dict

dict 是可迭代的，直接 `for...in` 遍历的是 **key**：

```python
d = {'name': 'Bob', 'age': 25, 'city': 'Beijing'}

for k in d:            # 遍历 key
for v in d.values():   # 遍历 value
for k, v in d.items(): # 遍历 key + value（最常用）
```

---

## 4. dict 视图对象

`.keys()` / `.values()` / `.items()` 返回的**不是 list**，而是视图对象：

```python
d = {'a': 1, 'b': 2, 'c': 3}

v = d.values()
type(v)  # <class 'dict_values'>
```

| 方法 | 返回类型 | 示例 |
|------|---------|------|
| `d.keys()` | `dict_keys` | `dict_keys(['a', 'b', 'c'])` |
| `d.values()` | `dict_values` | `dict_values([1, 2, 3])` |
| `d.items()` | `dict_items` | `dict_items([('a', 1), ('b', 2)])` |

视图的特点：**动态反映字典变化**：

```python
d['d'] = 4
print(v)  # dict_values([1, 2, 3, 4]) -- 自动更新了
```

视图能迭代、能判断包含，但**不能索引**：

```python
for x in v:     # OK
2 in v          # OK
v[0]            # TypeError!
```

需要 list 就手动转：`list(d.values())`

## 5. 判断对象是否可迭代

```python
from collections.abc import Iterable

isinstance('abc', Iterable)     # True
isinstance([1, 2, 3], Iterable) # True
isinstance(123, Iterable)       # False
```

底层原理是检查对象有没有 `__iter__` 方法：

```python
hasattr('abc', '__iter__')   # True
hasattr(123, '__iter__')     # False
```

常见类型的可迭代性：

| 类型 | 可迭代 | 遍历得到的是 |
|------|--------|-------------|
| str | Yes | 单个字符 |
| list | Yes | 元素 |
| tuple | Yes | 元素 |
| dict | Yes | key |
| set | Yes | 元素 |
| range | Yes | 数字 |
| int / float / bool | No | - |
| None | No | - |

## 6. collections.abc 抽象基类

`collections.abc` 提供一组**抽象基类（Abstract Base Classes）**，用来判断对象支持哪种"协议"（能力）。类似 Java 的 Interface 或 TypeScript 的 type/interface。

### 常用 ABC

| ABC | 要求实现的方法 | 典型类型 | 用途 |
|-----|--------------|---------|------|
| `Iterable` | `__iter__` | list, str, dict | 能 `for` 遍历 |
| `Iterator` | `__iter__` + `__next__` | generator, file | 能 `next()` 逐个取值 |
| `Sized` | `__len__` | list, dict, str | 能 `len()` |
| `Container` | `__contains__` | list, set, dict | 能 `in` 判断 |
| `Hashable` | `__hash__` | str, int, tuple | 能作为 dict 的 key |
| `Callable` | `__call__` | 函数, 类 | 能 `obj()` 调用 |
| `Sequence` | `__getitem__` + `__len__` | list, tuple, str | 能索引 + 有长度 |
| `Mapping` | `__getitem__` + `__len__` + `__iter__` | dict | 字典类接口 |
| `Set` | `__contains__` + `__iter__` + `__len__` | set, frozenset | 集合类接口 |

### 类型检查

```python
from collections.abc import Iterable, Callable, Mapping

isinstance([1, 2], Iterable)     # True
isinstance(len, Callable)        # True
isinstance({'a': 1}, Mapping)    # True
```

### 自定义类实现协议

```python
from collections.abc import Iterator

class Counter(Iterator):
    def __init__(self, n):
        self.n = n
        self.i = 0

    def __next__(self):
        if self.i >= self.n:
            raise StopIteration
        self.i += 1
        return self.i

for x in Counter(3):
    print(x)  # 1, 2, 3
```

### 关系层级

```text
Iterable          能遍历
  └─ Iterator     能逐个取值（一次性）

Container         能 in 判断
Sized             能 len()
Iterable          能遍历
  └─ Collection = Container + Sized + Iterable
       ├─ Sequence   有序，能索引（list, tuple, str）
       ├─ Set        无序，不重复（set）
       └─ Mapping    键值对（dict）
```
